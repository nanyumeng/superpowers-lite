# Changelog

All notable Superpowers Lite changes will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and versions follow Semantic Versioning with a `lite` prerelease identifier.

## [6.1.1-lite.1] - Unreleased

### Added

- Independent Superpowers Lite project identity and bilingual introduction.
- Preliminary GPT-5.6 upstream-versus-Lite evaluation report.
- Release metadata, privacy, security, support, contribution, and upstream-sync
  documentation.

### Changed

- Reduced the 14 core skill files from 3,322 to 1,053 lines while preserving
  the main workflow, skill names, specs/plans directories, TDD, and completion
  verification.
- Made single-agent execution the default and reserved delegation for work that
  benefits from isolation, parallelism, or explicit user direction.
- Replaced inherited package, plugin, marketplace, author, and repository
  metadata with Lite-owned values.
- Replaced inherited visual branding with a local Superpowers Lite identity.

### Removed

- The brainstorming visual companion's remote upstream logo request and
  version-bearing telemetry behavior.
- Inherited upstream sponsorship and community promotion from Lite-owned
  release surfaces.
