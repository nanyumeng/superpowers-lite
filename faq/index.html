Product

What is Superpowers Lite?

It is a complete skills-directory overlay for an existing Superpowers installation. It keeps the established development spine and makes planning, delegation, worktrees, review, and debugging conditional on scope and risk.

Is it a standalone plugin or a rewrite?

No. It deliberately reuses the upstream adapter and replaces only the behavior-shaping skills/ directory. It does not ship platform manifests, loaders, or hooks.

Is every file Markdown?

No. The current 48-file skills tree contains 36 Markdown files and 12 helper files used by the skills. The behavior rules are primarily reviewable Markdown, but “Markdown-only” would be inaccurate.

Is this official or endorsed?

No. Superpowers Lite is independently maintained and promoted by nanyumeng. It is not endorsed by Superpowers, OpenAI, or Anthropic.

Compatibility and safety

What stays compatible?

Skill names, the main brainstorm-to-verification workflow, specs and plans paths, TDD, systematic debugging, and completion verification remain. Read the comparison for the exact boundary.

Will it change my project?

The documented installation changes no application code, project configuration, AGENTS.md, CLAUDE.md, specs, or plans. It backs up and replaces the active Superpowers skills/ directory outside the project.

How do I Rollback?

Retain the exact timestamped backup created during installation. Follow the Rollback prompt to validate and restore it using recoverable moves.

What happens when upstream updates?

The upstream update may overwrite the overlay. Review the new upstream release, then rerun the Lite installation if you still want it, creating a new backup.

Evidence

Does Lite guarantee 30%+ fewer tokens?

No. That figure is a maintainer field observation from ongoing real-project work. It is not a controlled benchmark or a guarantee.

What does the 68.3% number mean?

It is the line-count reduction across 14 corresponding core SKILL.md files versus upstream v6.1.1: 3,322 lines to 1,053. It measures source text, not task cost or accuracy.

Is Lite more accurate?

That has not been established. An earlier three-task directional micro-test had mixed token results and did not prove an accuracy advantage. See Evidence and limitations.

Project

Who maintains it?

The GitHub user nanyumeng maintains and promotes Superpowers Lite. Superpowers was created by Jesse Vincent and its contributors.

Where should I report a problem?

Use the Superpowers Lite issue tracker for overlay-specific problems. Do not use upstream issue threads to promote the fork.

What is the license?

MIT. The repository includes LICENSE and NOTICE files with upstream attribution.
